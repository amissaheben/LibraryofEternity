# Institutional Roadmap

## Foundation v1.0 RC1

The architectural phase is frozen. Current work is restricted to repository implementation, graph population, validation, provenance, and release preparation.

## Platform milestones

### v0.1 — Repository browser

- Markdown and YAML ingestion
- archive browsing
- document reader
- search and registry
- health dashboard

### v0.2 — Dependency explorer

- interactive graph
- relationship panels
- class and status filters
- stronger graph validation

### v0.3 — Governed authoring

- document creation and editing
- workflow states
- Research Lock enforcement
- snapshots and change logs
- release controls

### v0.4 — Provenance

- lineage explorer
- amendment chains
- supersession and retirement maps
- trustee-grade audit trail

### v0.5 — Secure institutional platform

- authentication and sessions
- trustee roles and permissions
- PostgreSQL persistence
- Git-backed revision records
- review comments and decisions
- immutable activity logs
- Docker and production security

### Future releases

- v0.6 publication exports
- v0.7 repository-grounded research assistant
- v0.8 external audit portal
- v1.0 public institutional release

## Current sprint

**Sprint 1: Engineering baseline**

- repository documentation
- CI validation
- issue and pull-request governance
- platform source migration
- reproducible local setup
