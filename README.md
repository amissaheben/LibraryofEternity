# Library of Eternity

A constitutional research platform for modelling reality through auditable laws, principles, structures, thresholds, evidence, and governed inquiry.

## Current status

- **Foundation:** v1.0 RC1
- **Research Lock:** Active
- **AA-0006:** Constitutionally suspended
- **Platform:** Implementation phase

## Institutional architecture

The repository separates three domains:

1. **Layer A — The Universe:** objective constitutional claims and canonical volumes.
2. **Layer B — The Institution:** research, audits, adversarial reports, decisions, and evidence.
3. **Layer C — Meta-History:** institutional principles, trustee records, and the history of the Library itself.

## Repository layout

```text
apps/            Application entry points

foundation/      Canonical and constitutional records

scripts/         Validation and repository automation

```

## Development workflow

All implementation work begins on `development` or a `feature/*` branch and reaches `main` through reviewed pull requests.

```text
Research → Audit → Approval → Implementation → Validation → Release
```

## Governance

The Constitution governs contributors. Reduction has priority over expansion, repository integrity exceeds publication speed, and every permanent claim must possess provenance.
