# Foundation Repository

This directory is the constitutional source of truth for the Library of Eternity.

## Constitutional layers

- `AA/` — canonical universe volumes
- `LC/` — Library Constitution and evidence standards
- `FO/` — Foundation Observations
- `ADR/` — Architectural Decision Records
- `RP/` — Research Programs
- `AR/` — Adversarial Reports
- `IA/` — Internal Audits
- `IRN/` — Intermediate Research Nodes
- `IH/` — Institutional History
- `IP/` — Institutional Principles
- `TC/` and `TM/` — Trustee certifications and memoranda

## Repository rules

1. Markdown and YAML metadata are authoritative.
2. Every permanent record must possess provenance.
3. Retired and superseded records are preserved.
4. Research Lock remains active until the governing audit authorizes release.
5. No database may silently replace or mutate the constitutional source.
