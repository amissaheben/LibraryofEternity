# Contribution Protocol

1. Every artifact requires a unique ID and YAML front matter.
2. Every claim requires provenance.
3. Dependencies must reference registered IDs or protected terms.
4. Layer boundaries must not be violated.
5. Retired concepts are preserved, not deleted.
6. Canon changes require constitutional amendment.
7. Research artifacts cannot promote themselves into canon.
8. Research Lock prohibits new AA canon.
